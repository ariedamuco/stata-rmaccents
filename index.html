rmaccents Stata Package

This site hosts the rmaccents package for Stata.

How to Install

net install rmaccents, from("https://ariedamuco.github.io/stata-rmaccents/")

About the Package

The rmaccents package is designed to remove accented characters from text variables in Stata, making it easier to process data with accented text.

Features

  • Supports multiple accents and diacritics.
  • Options for replacing original variables or creating new ones.

Examples

Replace accented characters in the original variable:

. rmaccents name, replace

Create a new variable without accents:

. rmaccents city, newvar(city_noaccent)
